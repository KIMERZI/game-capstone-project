사용 방법:
	1. 시작화면
	- space를 누르면 시작
	
	2. 게임 화면
	- 귀신을 피해서 도망가야함. 귀신에 닿으면 사망
	- 귀신은 10초마다 하나씩 늘어남, 속도도 빨라짐
	- 미니맵: 왼쪽 위 미니맵을 통해 귀신 위치 확인 가능
	- 타이머: 
	
	3. 조작법
	- w:앞쪽 a:왼쪽 s:뒤쪽 d:오른쪽 으로 이동
	- 마우스 커서 위치: 마우스 커서가 오른쪽/왼쪽 끝에 위치하면 시선이 오른쪽/왼쪽으로 회전함
	- 마우스 우클릭: 총 발사, 타겟에 적중하면 타겟이 멈춤
	- esc를 눌러 pause, 다시 esc를 누르면 재개
	- c: 치팅, 귀신을 모두 없앰
	
	4. 게임 종료
	- 귀신에 닿아서 사망하면 종료 화면이 뜸
	- esc를 누르면 게임 완전 종료, space를 누르면 게임이 재시작함
